# 项目最终阶段进度说明

## 项目目标

本项目面向 C++ 实训任务中的“矩阵、向量与线性方程组”方向，目标是实现一个基于 C++17 的线性代数计算库。项目以模板、类、继承、多态、运算符重载和异常处理为主要技术点，提供向量、矩阵、复数、有理数、高精度整数和高精度浮点数等基础能力。

当前阶段已在任务一、任务二基础上继续完成任务三的核心功能：线性方程组求解、非方阵最小二乘、主特征值/特征向量求解和分块矩阵乘法。

## 当前完成情况

CMake 已配置 interface library：

- `Utils`

并提供 `test` 可执行目标用于演示。

### 2. 基础代数类型

已实现以下类型：

- `ComplexNumber<T>`：模板复数类型，支持复数加、减、乘、除、比较和流输入输出。
- `RationalNumber<T>`：模板有理数类型，支持自动约分、分母规范化、四则运算和比较。
- `BigInteger<>`：高精度整数类型，支持任意长度有符号整数的加、减、乘、除、取模、比较和流输入输出。
- `HighPrecisionNumber<>`：高精度浮点/定点小数类型，支持字符串、整数、`float`、`double` 构造，支持四则运算、比较和流输入输出。

其中 `HighPrecisionNumber<>` 已增加非 `explicit` 的 `float` 和 `double` 构造函数，方便如下写法：

```cpp
HighPrecisionNumber<> x = 1.25;
HighPrecisionNumber<> y = 2.5f;
```

### 5. 任务二范数与条件数

已完成任务二的核心要求：

- 矩阵 Frobenius 范数。
- 矩阵行和范数。
- 矩阵列和范数。
- 矩阵谱范数。
- 矩阵条件数。

谱范数当前使用幂迭代法近似计算最大奇异值。条件数当前使用行和范数：

```text
cond(A) = ||A|| * ||A^{-1}||
```

当矩阵不可逆或不是方阵时，`condition_number()` 返回 `infinity`，符合任务书要求。

### 6. 任务三线性方程组与特征值

已新增 `Matrix<T>` 的任务三接口：

- `solve(b)`：求解方阵线性方程组 `Ax=b`，使用带主元选择的高斯消元和回代。
- `least_squares(b)`：求解非方阵最小二乘解，采用正规方程 `A^T A x = A^T b`。
- `dominant_eigenpair(max_iterations, tolerance)`：使用幂迭代法求实方阵主特征值和对应单位特征向量。
- `block_multiply(matrix, block_size)`：实现分块矩阵乘法，可与普通矩阵乘法进行结果和性能对比。

`src/final_demo.cpp` 已增加对应演示：

- 三元一次方程组求解。
- 过定约束直线拟合的最小二乘系数。
- 对称矩阵主特征值与特征向量。
- 普通矩阵乘法和分块矩阵乘法的一致性与耗时对比。

### 7. 异常处理

已在主要非法操作中加入异常处理：

- 向量维度不匹配时抛出异常。
- 矩阵加减乘维度不匹配时抛出异常。
- 非方阵求行列式、逆矩阵或伴随矩阵时抛出异常。
- 奇异矩阵求逆时抛出异常。
- 奇异方阵求唯一线性方程组解时抛出异常。
- 最小二乘右端向量维度不匹配时抛出异常。
- 非方阵求特征值时抛出异常。
- p-范数中 `p < 1` 时抛出异常。
- 高精度数值除以 0 时抛出异常。
- 文件打开失败时抛出异常。

## 风险与限制

当前实现仍有以下限制：

- 谱范数为幂迭代近似值，不是符号精确结果。
- 主特征值为幂迭代近似值，目前不返回全部特征值。
- 最小二乘当前使用正规方程，病态矩阵上稳定性不如 QR 分解。
- 高精度浮点采用定点小数模型，除法默认保留 30 位小数。
- 分块矩阵乘法提供基础优化和对比入口，尚未实现多线程或 Strassen 算法。
- 当前演示程序主要覆盖核心接口，后续仍需要补充自动化测试。
